



# 微信H5，判断是否在微信浏览器中打开

一进来时判断

```jsx

var ua = navigator.userAgent.toLowerCase();
var isWeixin = ua.indexOf('micromessenger') != -1;
if (!isWeixin) {
    window.location.href = 'https://open.weixin.qq.com/connect/oauth2/authorize?appid='+this.$appid;  // 自己的appid

    // 也可以跳转到自己写的错误页面
    // uni.reLaunch({
    //  url:'/pages/address/address'
    // })
}
```
