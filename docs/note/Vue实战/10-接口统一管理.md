10、代码改变时实现页面自动刷新
根目录下vue.config.js文件设置

```js
module.exports = {
    //关闭eslint
    lintOnSave: false,
    devServer: {
        // true 则热更新，false 则手动刷新，默认值为 true
        inline: true,
        // development server port 8000
        port: 8001,
    }
```


注意：修改完该配置文件后，要重启一下项目
