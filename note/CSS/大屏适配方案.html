    大屏适配方案

    一、大屏适配方案

    1. px 转 rem
      • 页面中有其他已经开发有其他的元素有导航文字, 设置fontSize时会影响到页面中已经写好的内容
      • 页面中使用了 echarts 图表, 里面的参数没办法应用 rem 的比例
    2. viewpoint--移动端 +
    3. 媒体查询 @media
      • 书写大量媒体查询代码，较繁琐；
      • 无法保证完全兼容所有的屏幕
      • 无法支持 echarts 图表中的参数进行适配
    4. 计算屏幕缩放比 动态设置像素值
